Two-Mass Spring Vibration System

Eigenvalues • Natural Frequencies • Mode Shapes

1. Mathematical Model

Equation of motion:

[M]{ẍ} + [K]{x} = 0

Assuming harmonic motion:

{x} = {X}e^{iωt}

This leads to the eigenvalue problem:

([K] − ω²[M]){X} = 0

2. Physical Meaning

  • Eigenvalues → ω² (natural frequencies squared)
  • Eigenvectors → mode shapes
  • Mode 1: masses move together
  • Mode 2: masses move opposite
